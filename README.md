# AI Resume Analyzer 

This is a web-based application that analyzes resumes using Python and provides insights such as:

## Features
- Extracts skills from resume
- Predicts suitable job role
- Suggests missing skills
- Provides resume score
- Gives detailed feedback

## Tech Stack
- Python
- Flask
- HTML, CSS

## How to Run
1. Install dependencies:
   pip install flask PyPDF2

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
